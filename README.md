# LB — Personal Claude Bot Build

This repository documents the planning, development, and progress of LB, my personal Claude-powered automation bot.

LB is intended to become a personal AI assistant that helps with resale alerts, information filtering, trading research, email, scheduling, weekly tee times, and other daily automation tasks.

## Purpose

The purpose of this repository is to track the build process, document project goals, organize ideas, and show steady progress as LB is developed.

## What LB Is Intended To Do

- Filter important information from noisy sources
- Help with resale decisions
- Support trading research
- Read and summarize useful information
- Help manage email and scheduling
- Help set weekly tee times
- Automate repetitive daily tasks
- Send alerts when something needs attention

## Repository Structure

- `PROJECT_GOALS.md` — overall goals for LB
- `ROADMAP.md` — planned build phases
- `DAILY_LOG.md` — daily progress journal
- `notes/ideas.md` — rough ideas and future features
