# Technical Information

## Hardware

The main hardware for this project is a MINISFORUM UM880 Plus Mini PC.

Current hardware specs:

- MINISFORUM UM880 Plus Mini PC
- AMD Ryzen 7 8845HS processor
- 8 cores / 16 threads
- Up to 5.1 GHz
- 32GB DDR5 RAM
- 512GB SSD
- Radeon 780M graphics
- 2.5G LAN
- WiFi 6E
- USB4 support
- Multiple display outputs through HDMI, DisplayPort, and USB4
